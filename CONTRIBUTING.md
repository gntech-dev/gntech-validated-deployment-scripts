# Contributing

A contribution is eligible only when all of the following are true:

1. The exact workflow was executed successfully in a real lab.
2. The script has no secrets, customer identifiers, private topology, or organization-specific defaults.
3. Required credentials are requested securely at run time.
4. The script is idempotent or explicitly detects and stops on unsafe drift.
5. PowerShell parser validation passes.
6. The pull request records the lab evidence and the source revision.

Do not publish scripts that were only reviewed, simulated, or partly tested.
